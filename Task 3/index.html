CSS Selectors Exercise

This exercise will test your understanding of CSS selectors.

Follow the instructions in the comments of the CSS section to complete the exercise.

Make sure to use universal, element, ID selector, class selector, and grouping selector where necessary.

Instructions:

  • Select all paragraphs and change their text color to blue.
  • Select the element with ID "header" and change its font size to 24px.
  • Select all elements with the class "highlight" and change their background color to yellow.
  • Select all <h1> and <h2> elements and center-align their text.
  • Combine the selector for paragraphs and the class selector ".highlight" into one rule to make their text bold.
